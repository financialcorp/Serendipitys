# 🛥️ Serendipity Yacht Charter

**Refined Luxury on the Florida Gulf Coast**

Serendipity is a premium web application designed for a 94' Lazzara Hardtop motor yacht based in Saint Petersburg, Florida. This platform provides an immersive experience into the world of high-end yachting, featuring seamless booking inquiries, video showcases, and elegant design.

## ✨ Features

- **Immersive Hero Section**: Captivating imagery and a direct call-to-action to watch the onboard experience video.
- **Interactive Availability Calendar**: A custom-built calendar allowing guests to check available dates for their next voyage.
- **Cinematic Video Integration**: Integrated Vimeo player showcasing the vessel's luxury and Gulf Coast excursions.
- **Experience Showcases**: Detailed sections for sunset cruises, corporate events, and wellness retreats at sea.
- **Luxury Accommodations**: Highlighting the master stateroom and VIP suites with spa-inspired amenities.
- **Scroll Reveal Animations**: Smooth, motion-driven transitions powered by Framer Motion for a premium feel.
- **Mobile-Responsive Design**: Fully optimized for exquisite viewing on any device, from smartphones to desktops.
- **Direct Contact Integration**: Easy access to the charter team and physical location at Maximo Marina.

## 🛠️ Technical Stack

- **Framework**: [React 18](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Seraphic serif headings paired with clean, modern sans-serif body text for a "Quiet Luxury" aesthetic.

## 🚀 Getting Started

### Local Development

1. **Clone the repository**:

   ```bash
   git clone <repository-url>
   cd serendipity-yacht
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Start the development server**:

   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to `http://localhost:3000` to view the application.

## 📁 Project Structure

- `src/App.tsx`: Main application architecture and component definitions.
- `src/index.css`: Global styles and Tailwind configurations.
- `src/components/`: Reusable UI elements (Modals, Navigation, Form components).
